# Proyecto Shiny Comparativa Los Simpsons vs South Park

Para el desarrollo de este proyecto se requirieron archivos de datos para poder obtener información, dentro de este repositorio
ya se encuentran los datos limpios listos para desplegar la aplicación, sin embargo, los datos archivos de datos originales al tener
un peso elevado no fue posible anexarlos, por lo que se pueden obtener de los siguientes enlaces:
- [series.ratings.tsv.gz](https://datasets.imdbws.com/title.ratings.tsv.gz)
- [series.episode.tsv.gz](https://datasets.imdbws.com/title.episode.tsv.gz)
- [series.basics.tsv.gz](https://datasets.imdbws.com/title.basics.tsv.gz)

  Como aclaración estos deben ser guardados en la carpeta data/raw con el mismo nombre que contienen los enlaces.
